KLP Image Importer

Automates the import of SEM images into KLayout using transformation parameters from .klp files.
How it works

This macro uses the .klp files generated by the software from Gilles Nogues. It automatically applies the transformation parameters (center, scale, rotation, and shear) so you don't have to enter them manually in KLayout.
Installation

    In KLayout, go to Tools > Manage Packages.

    Click Settings (Gear Icon) > Repositories tab > + (Add).

        Name: Lab Packages

        URL: https://github.com/ShashankArunkumar/Klayout_Packages.git

    Go to Install New Packages tab, find KLP Image Importer, and click Apply.

Usage

    Shortcut: Press Ctrl+Shift+I.

    Requirement: The .klp file must be in the same directory as the SEM image and have the same filename.

    Process: Select your SEM image, and the macro will instantly apply the correct transformation.

Note: If images are not visible after import, ensure View > Show Images is enabled.
